Lesson 3 Notes

CSS stands for Cascading Style Sheets with cascading being the most important part of the acronym. Cascading eliminates repetition which saves time and prevents typing errors when adding style to elements.
Some commonly used CSS Properties include (for text) text-align, color, font-family, font-size, font-weight, font-style, etc., and (for elements) background-color, border-radius, padding, margin, width, etc.
